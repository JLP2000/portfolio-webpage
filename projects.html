Upper Hand Poker

Project Description:

Upper Hand Poker is a group project developed a real-time multiplayer poker game web application

But it isn't regular poker... it has a twist: at the beginning of each game, users can click on one other user at the table to view their hand for the rest of that game, providing players with an extra bit of information to WIN BIG and LOSE LITTLE

It was built using Vite on the frontend and Python on the backend with a SQL database for user login storage and Redux when dealing with the gamestate on the frontend .

This project uses the "flask-socketio" library to provide the functionality that allows users to create/join games and play an online game of poker with friends!

URL Shortener

Project Description:

URL Shortener is a tool that allows users to input a link that they want to shorten to send to others, making it easier to share or save. It also stores the history of a users converted links as well as a QR code that can be sent

This was a pair programming project where we were tasked to build a URL shortener web-app using Python, we decided to use the Flask framework where we build the API that accessed our SQL database that stored the links that users have inputted and the shortened version of that link

For the testing side, we used pytest to test the different routes of our API and get a good amount of coverage of our code

Github User Searcher

Project Description:

The Github User Searcher is a tool that lets users search-up existing Github users and have all their stats displayed to them on a page with graphs and images. Some of the stats include:

  • % of languages used
  • top repository
  • stats of each of their repositories

This was a pair programming project to build an web application that allows users to search up the statistics of Github users displayed in a visually-appealing way

This project was built using Vite to design the webpage and JS to provide the functionality behind the webpage

We decided to explore Vite during this project as we felt we had a good grasp of React and wanted to explore an alternative build tool.

Habitual

Project Description:

Habitual was a group project that developed a habit-tracker web application for users to track habits and build streaks to help users develop good habits in their daily lives

With Habitual, users can create accounts and create personal habits that they can track daily, weekly or monthly. Habitual aims to assist users in developing habits they want to build

This project was built using HTML/CSS on the frontend and a Node.js backend with a SQL database for storage of created habits and streaks along with their user information

This project mainly focused on building and initialising a SQL database with an application, ElephantSQL was used to host the database that would store user information along with habits and streaks that belong to each user
